Welcome to BaseballDB!

BaseballDB is your API for all things baseball statistics. The data is primarily based on Lahman's Baseball Database with some adaptations. An example application using the API can be found here.

Using the API

The base URL for the API is http://baseballdb.online/api/.

Endpoints

GET: /players

Returns list of players.

Request Parameters
ParameterDescription
pPage Number
orderField to order results by. Defalt: namelast

GET: /player/:playerid

Returns player personal data as well as statistics by year.

/player/search

Return players by matched to query by first or last name.

Request Parameters
ParameterDescription
qQuery String

GET: /teams

Request Parameters
ParameterDescription
pPage Number, default: 1
orderField to order results by. Defalt: year

GET: /team/:teamid

Returns team data. Teamid points to a specific team for a specific year.

GET: /franchise/:franchid

Returns all teams from a given franchise.

GET: /baberuth/:playerid

Returns distance from Babe Ruth for given player and information about the links in ordered lists.
